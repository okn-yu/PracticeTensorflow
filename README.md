# TensorFlowの概要
* TensorFlowはデータフローグラフを利用した数値計算のためのオープンソース・ソフトウェア・ライブラリ
* ディープラーニングに用途が限定されているわけではないことに注意

# Define and Run
* 手続きを定義し結果が必要になる毎にrunを実施
* 実行前に事前にNWを定義する必要がある
* 一方Define by Runでは実行時にNWの変更が可能
